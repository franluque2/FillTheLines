# FillTheLines

A small personal project for recording and uploading voice lines for small projects.

## Features

- Record voice lines
- Upload voice lines
- Simple and lightweight

## Setup

1. Copy `config.example` to `config.json`
2. Install dependencies:
    ```bash
    npm install
    ```
3. Start the project:
    ```bash
    npm start
    ```

## Notes

- No account management included
- Designed for personal use only
